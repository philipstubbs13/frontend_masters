# Audits

## What can slow down a website?

- Back End
  - Database
  - Server
  - Network
  - Compression
- Front End
  - Large bundle
  - Too many files
  - Images or JavaScript not async
  - Images too large
  - Images uncompressed
  - Unused JavaScript or CSS
  - CSS in Document Body
  - Not using browser caching
